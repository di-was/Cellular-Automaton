# Cellular-Automaton
C implementation of [Conway's game of life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) using [SDL2](https://www.libsdl.org/) library.


# Key mapings
  ```-esc            ->  Terminate Window
     -r              ->  resume simulation
     -p              ->  pause simulation
     -right_click    ->  Reproduce cell
  ```
